This Solution contains multiple project that shows the basics of Python.
Most of the Project implemented require no user input and are made for demonstration and academic purposes.

This solution Contains the following projects:
 1.	1_Basics  		- Print, variables, strings, and casting
 2.	2_Operators   	- Arithmatic, Comparison, Logical, and Identity Operators
 3.	3_Conditionals  - If, Else, and Elif
 4.	4_Functions  	- User-Defined Functions and some Python Standard Functions
 5.	5_Sequences  	- Lists, Tuples, Dictionaries
 6.	6_Itterations 	- For Loop, While Loop, and Range()
